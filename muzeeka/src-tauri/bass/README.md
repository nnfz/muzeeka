# BASS Audio Library

Place the BASS library files here:

## Required
- `bass.dll` — Main BASS library

## Optional Addons  
- `bassflac.dll` — FLAC support
- `bassopus.dll` — Opus support  
- `basswma.dll` — WMA support
- `bass_aac.dll` — AAC/M4A support
- `bassmix.dll` — Mixer addon
- `bassfx.dll` — Effects addon

Download from: https://www.un4seen.com/
